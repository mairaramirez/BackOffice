# BackOffice Frontend - Vista Servicios

Este proyecto contiene SOLO el FRONTEND de la vista "Servicios" de un BackOffice, creado con Vite + React (JavaScript).

## Requisitos
- Node.js 18+

## Instalación y ejecución
1. Instalar dependencias:
   ```bash
   npm i
   ```
2. Iniciar el servidor de desarrollo:
   ```bash
   npm run dev
   ```
3. Abrir el navegador en la ruta:
   - http://localhost:5173/servicios

## Scripts
- `npm run dev` — inicia Vite en modo desarrollo
- `npm run build` — genera la build de producción
- `npm run preview` — sirve la build localmente
